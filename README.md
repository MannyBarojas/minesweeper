# Minesweeper

A classic 8×8 Minesweeper game playable in any browser — desktop or mobile. No dependencies, no build step. Open `index.html` and play.

## How to play

- **Reveal a cell:** tap or left-click
- **Flag a mine:** long-press (mobile) or toggle Flag Mode then tap; right-click on desktop
- Clear all non-mine cells to win. Hit a mine and it's game over.

## Rules

- 8×8 grid, 10 mines
- First click is always safe — mines are placed after your first tap
- Numbers show how many mines are adjacent to that cell
- 🚩 Flag cells you think are mines to keep track

## Tech

Single self-contained HTML file. Vanilla JS, no frameworks, no dependencies.
